Дополнительная практика

1. Создайте пакет main
2. Создайте класс Main и метод main
3. Создайте новый пакет taks
4. Создайте класс Test
5. Создайте переменную с модификатором public
6. Создайте внутри пакета task подпакет sub_task
7. Создайте в пакете sub_task класс SubTest
8. Создайте переменную с модификатором private
9. Создайте экземпляры классов Test и SubTest в классе
Main
10. Попробуйте изменить модификаторы классов и
переменных
11. Проанализируйте написанный код
